Introduction

Welcome to RoyaleForge — the most advanced open-source Clash Royale analytics platform. RoyaleForge combines machine learning, data analysis, and an intuitive interface to help players build winning decks and master the meta.

This documentation covers everything you need to know — from installation to advanced features. Whether you're a casual ladder player or competitive tournament pro, RoyaleForge provides the insights you need to climb higher.

Open Source & Free
RoyaleForge is 100% free and open-source under the MIT license. Contribute on GitHub to help improve the platform.

Installation

RoyaleForge runs locally on your machine. You'll need Python 3.10 or higher installed.

Requirements

  • Python 3.10 or higher
  • pip (Python package manager)
  • Git for cloning the repository
  • Modern web browser (Chrome, Firefox, Edge)

Clone the Repository

Terminal 
# Clone the project
git clone https://github.com/ShauryaSingh1709/Clash-Royale.git
cd Clash-Royale

# Install dependencies
pip install -r requirements.txt

Quick Start

Get up and running in less than 2 minutes with these steps.

1
Generate the Dataset
Run the dataset generator script to create the necessary data files.
Terminal 
python scripts/generate_dataset.py
2
Start the Server
Launch the Flask backend. ML models will load automatically.
Terminal 
python -m backend.app
3
Open in Browser
Navigate to http://127.0.0.1:5000 in your browser. You're ready to analyze decks!
You're all set!
Visit the Deck Analyzer to start building your first AI-analyzed deck.

Deck Analyzer

The Deck Analyzer is the heart of RoyaleForge. Build any 8-card deck and receive instant AI-powered analysis.

How It Works

  • Select 8 cards from the library (97+ available)
  • ML models predict your deck's win rate
  • Receive a strength score (0-100) with letter grade
  • Get classified into one of 5 archetypes
  • See suggested card replacements to improve
  • Discover similar high-performing decks

Card Explorer

Browse all available Clash Royale cards with detailed statistics. Filter by rarity, type, or sort by win rate and usage.

Meta Dashboard

Visualize the current meta with four interactive charts powered by Chart.js. Track popular cards, top win rates, archetype distribution, and seasonal trends.

ML Models Overview

RoyaleForge uses four production-grade machine learning models trained on real Clash Royale data. All models load into memory at startup for instant predictions.

Making Predictions

All predictions happen server-side in milliseconds. The frontend communicates with the backend via REST API.
