ABOUT CR ANALYZER

🎯 Mission

Clash Royale Deck Analyzer is an open-source platform that helps players master the game through data-driven insights and AI-powered analysis. Our goal is to make competitive analytics accessible to every player.

🛠️ Tech Stack

Frontend

HTML5, CSS3, JavaScript, Chart.js

Backend

Python, Flask, REST API, OOP

ML & Data

Scikit-Learn, Pandas, NumPy, KNN

🤖 ML Models

  • 🎯 Win Rate Predictor - Random Forest Regressor
  • 💪 Deck Strength Scorer - Gradient Boosting
  • 🏷️ Archetype Classifier - Random Forest Classifier
  • 🔍 Similar Deck Finder - KNN + Cosine Similarity

⭐ Open Source

This project is open-source and welcomes contributions from the community.
