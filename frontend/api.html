Authentication

The RoyaleForge API is currently open and requires no authentication. All endpoints are accessible without API keys.

Rate Limits
No rate limits are enforced in the local version. For production deployments, consider implementing rate limiting.

Response Format

All API responses are in JSON format and include a success field indicating whether the request was processed successfully.

JSON Response 
{
  "success": true,
  "data": { ... },
  "total": 42
}

Error Handling

Errors return a success: false with an error message and HTTP status code.

Error Response 
{
  "success": false,
  "error": "Deck must contain exactly 8 cards",
  "code": 400
}

Cards API

GET /api/v1/cards

Returns all cards in the database. Supports optional filters via query parameters.

Query Parameters
Parameter Type Description
rarity string Filter by rarity (Common, Rare, Epic, Legendary, Champion)
type string Filter by type (Troop, Spell, Building)
min_elixir integer Minimum elixir cost
max_elixir integer Maximum elixir cost
Example Request
cURL 
curl http://127.0.0.1:5000/api/v1/cards?rarity=Legendary
GET /api/v1/cards/{name}

Returns detailed information about a specific card by name.

cURL 
curl http://127.0.0.1:5000/api/v1/cards/Hog%20Rider
GET /api/v1/cards/search

Search cards by name with partial matching.

Query Parameters
ParameterTypeDescription
q string Search query (required)
GET /api/v1/cards/top/win-rate

Returns the top N cards sorted by highest win rate.

GET /api/v1/cards/top/usage

Returns the top N cards sorted by highest usage rate.

Decks API

POST /api/v1/decks/analyze

Performs complete deck analysis with ML predictions, strengths, weaknesses, and full scoring breakdown.

Request Body
JSON 
{
  "cards": [
    "Hog Rider", "Musketeer", "Ice Spirit", "Skeletons",
    "Fireball", "The Log", "Cannon", "Ice Golem"
  ]
}
POST /api/v1/decks/validate

Validates a deck without performing analysis. Checks size, duplicates, and card validity.

POST /api/v1/decks/score

Returns ML predictions: win rate, strength score, and archetype classification.

Meta API

GET /api/v1/meta/summary

Returns overall meta statistics, averages, and totals.

GET /api/v1/meta/popular

Returns most popular cards by usage rate.

GET /api/v1/meta/archetypes

Returns archetype distribution and performance metrics.

Recommendations

POST /api/v1/recommend/full

Returns complete recommendation report including ML predictions, similar decks, and improvement suggestions.

POST /api/v1/recommend/similar

Finds decks similar to the provided deck using KNN + Cosine similarity.
