Building Your First Winning Deck

This tutorial walks you through building a competitive deck from scratch using RoyaleForge's AI-powered Analyzer.

1
Choose Your Win Condition
Start with a strong win condition card. Popular choices include Hog Rider, Giant, Balloon, or X-Bow. This card defines your deck's strategy.
2
Add Support Troops
Add 2-3 support troops that complement your win condition. For Hog Rider, consider Musketeer, Ice Golem, or Valkyrie.
3
Include Defense
Add defensive cards or buildings. Cannon, Tesla, or Inferno Tower are great defensive options.
4
Pick Your Spells
Every deck needs 2-3 spells. Combine a big spell (Fireball, Lightning) with a small spell (The Log, Zap) for versatility.
5
Add a Cycle Card
Include a cheap cycle card like Skeletons, Ice Spirit, or Goblins to maintain elixir advantage.
6
Analyze & Optimize
Click "Analyze Deck" to get AI predictions. Review the strengths and weaknesses, then apply suggested card replacements to improve your deck.
Pro Tip
Aim for an average elixir cost between 3.0 and 4.0 for a balanced deck. Lower for cycle decks, higher for beatdown.

Reading the Meta Dashboard

The Meta Dashboard reveals what's working in the current game state. Use these insights to make smarter deck choices.

1
Identify Top Cards
Check the "Most Used Cards" chart to see what's dominant. Either include these cards or build counters to them.
2
Find Hidden Gems
Cards with high win rates but low usage are underrated. They can give you a competitive edge against unprepared opponents.
3
Study Archetypes
The archetype distribution shows what styles are popular. Build a deck that counters the dominant archetype.

Using the API in Your Project

Integrate RoyaleForge into your own application using the REST API. Here's how to analyze a deck from JavaScript.

JavaScript Example

JavaScript 
// Analyze a deck using the RoyaleForge API
async function analyzeDeck(cards) {
    const response = await fetch(
        'http://127.0.0.1:5000/api/v1/decks/analyze',
        {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ cards })
        }
    );
    return await response.json();
}

// Example usage
const myDeck = [
    'Hog Rider', 'Musketeer', 'Ice Spirit', 'Skeletons',
    'Fireball', 'The Log', 'Cannon', 'Ice Golem'
];

analyzeDeck(myDeck).then(result => {
    console.log('Win Rate:', result.scoring.win_rate);
    console.log('Grade:', result.scoring.strength.grade);
});

Python Example

Python 
import requests

def analyze_deck(cards):
    response = requests.post(
        'http://127.0.0.1:5000/api/v1/decks/analyze',
        json={'cards': cards}
    )
    return response.json()

deck = [
    'Hog Rider', 'Musketeer', 'Ice Spirit', 'Skeletons',
    'Fireball', 'The Log', 'Cannon', 'Ice Golem'
]

result = analyze_deck(deck)
print(f"Win Rate: {result['scoring']['win_rate']}")
